## Overview
RR-14 adds Prettier to the project to enforce consistent code formatting across both the Vue frontend and Express backend. A shared config at the repo root applies the agreed rules — single quotes, semicolons, 4-space tabs, and ES5 trailing commas — so all code is formatted uniformly regardless of editor.
