# Plan: RR-13

## Work Item

**Title:** Scaffold Express server

## Overview

RR-13 introduces an Express server that will act as a proxy between the Rheanne Retriever Vue frontend and the Microsoft Foundry agent API. The server lives in a `server/` directory alongside the existing frontend code. This branch covers scaffolding the server, environment variable setup, and CORS configuration — Foundry integration follows in a subsequent branch.
