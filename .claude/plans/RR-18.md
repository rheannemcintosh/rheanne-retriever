# Plan: RR-18

## Work Item
**Title:** Add vueIndentScriptAndStyle Prettier rule

## Overview
RR-18 adds `vueIndentScriptAndStyle: true` to the Prettier config so that `<script>` and `<style>` block content in Vue SFCs is indented consistently with the rest of the codebase. Once the rule is in place, all existing Vue files are reformatted to apply it.
