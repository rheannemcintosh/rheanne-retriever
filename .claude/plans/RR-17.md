### Phase 3: Wire up API integration
Connect `ChatInput` submissions to the Express `/api/chat` endpoint and display the agent's response in `ChatWindow`.

- [ ] Create `src/composables/useChat.js` — manages the `messages` array and a `sendMessage(text)` function that POSTs to `http://localhost:3000/api/chat` and appends both the user message and agent reply to the list.
- [ ] Add a `VITE_API_URL` env var (`.env` and `.env.example` at the repo root) so the API base URL is configurable.
- [ ] Wire `useChat` into `App.vue` and pass `messages` and `sendMessage` down to the components.
